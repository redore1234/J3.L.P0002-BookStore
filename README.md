# Welcome to redore1234 repository

- Uploading image 

- Using data source into Java Web App

### How to use this?
It's required some libraries

 - jstl-api.jar
 - jstl-impl.jar
 - log4j-1.2.17.jar
 - sqljdbc42.jar
